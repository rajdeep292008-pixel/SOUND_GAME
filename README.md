## Simon Memory Master
A professional, production-quality browser-based Simon memory game built with pure HTML, CSS, and JavaScript. This project focuses on a premium user experience, glassmorphism design, and high-performance Web Audio API integration.

---

## 🚀 Overview
Simon Memory Master is a modern reimagining of the classic electronic memory game. The game challenges players to replicate increasingly complex sequences of colors and sounds, testing their cognitive recall and reflexes.

---

### 🎮 Core Features
1. Zero Dependencies: Built using vanilla JavaScript, CSS3, and HTML5.
2. Web Audio API: Generates high-quality, synthetic tones programmatically—no external audio files required.
3. Responsive Design: Optimized for desktop, tablet, and mobile devices using CSS Grid and fluid layouts.
4. Persistence: High scores are automatically saved to your browser's localStorage.
5. Accessibility: Full support for both mouse/touch interactions and keyboard inputs (R, B, G, Y keys).
6. Premium Aesthetic: Modern dark theme featuring glassmorphism, neon glow effects, and smooth CSS animations.

---

### 🕹️ How to Play
1. Start the Game: Click the "Start Game" button.
2. Watch: Observe the sequence of colored pads as they light up and play a unique tone.
3. Repeat: Click the pads (or press the corresponding keys) in the exact order shown.
4. Advance: Each successful round increases the sequence length and difficulty.
5. Game Over: The game ends if you miss a note in the sequence. Try to beat your personal high score!

 ### 🛠️ Technical Stack
1. HTML5: Semantic structure for game components.
2. CSS3: Flexbox/Grid for layout, glassmorphism, and hardware-accelerated animations.
3. JavaScript (ES6+): Logic managed via state-based triggers and asynchronous timing for smooth gameplay.
4. Web Audio API: Real-time synthesis for responsive, low-latency audio feedback.
